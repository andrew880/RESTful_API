# RESTful_API

## Quick Start

__0. Requirements__

    pip install -r requirements.txt
    
__1. Download the code to your computer.__
    
    git clone https://github.com/andrew880/RESTful_API.git
    
__2. Run API SERVER__
    
    python run.py
    
__3. Run main__

    python main.py
